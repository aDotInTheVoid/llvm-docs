bool AAValueSimplifyFloating::handleCmp ( Attributor &  A,
CmpInst &  Cmp 
)
inline

Check if Cmp is a comparison we can simplify.

We handle multiple cases, one in which at least one operand is an (assumed) nullptr. If so, try to simplify it using AANonNull on the other operand. Return true if successful, in that case SimplifiedAssociatedValue will be updated.
